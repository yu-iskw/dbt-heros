# Plugin Table Template

Use the following format for the plugin table in the root `README.md`.

```markdown
| Category    | Plugin                      | Description    | Status                    |
| :---------- | :-------------------------- | :------------- | :------------------------ |
| ${category} | [${name}](./plugins/${dir}) | ${description} | ${status_emoji} ${status} |
```
